# Automatic detection

## Introduction

By default, the extension automatically scans the current web page for iframe and frame tags every time you navigate to a new page or switch tabs, and updates the icon colour accordingly.

If you want to avoid automatic detection, you can disable this option. When it's disabled, you'll need to click the extension icon to run a scan.

## Icon colours

When automatic detection is disabled, the extension icon on new pages turns **blue** to indicate that the page has not been scanned yet. Once you click the extension icon, the scan runs automatically and the icon updates to reflect the result.

## Configuration

Required steps:

1. Click the `Configuration` button to open the configuration menu.
2. Deactivate the `Automatic detection` option.

## Behaviour when disabled

- Navigating to a new page or switching tabs will **not** trigger a scan.
- On new pages the icon turns **blue** to indicate the page has not been analysed.
- When you click the extension icon, a scan runs automatically and the icon updates.
