# Always show tags info

## Introduction

The information displayed when you click the `Show tags info` button can be configured to always be visible.

This avoids having to click on the `Show tags info` button.

## Configuration

Required steps:

1. Click the `Configuration` button to open the configuration menu.
2. Activate the `Always show tags info` option.
